
# 🖐🏻🤓🖐🏻 Fucker Bot – ربات فحاشی هدفمند

---

## 🎯 معرفی پروژه  

ربات فاکر با هدف ایجاد یک ابزار **سرگرم‌کننده و مدیریتی** برای گروه‌های تلگرام طراحی شده است.  
کاربران خاص (سودوها) می‌توانند با استفاده از یک **پنل مدیریتی ساده**، تارگت (شخص مورد نظر) را مشخص کرده و به او فحش‌های مختلف (به صورت **متن یا ویس**) ارسال کنند.  

---

## ✨ قابلیت‌های کلیدی  

- 🎯 **تغییر تارگت** (شخص مورد نظر برای فحاشی)  
- ➕ **اضافه کردن سودوهای جدید** با آیدی عددی  
- ➕ **اضافه کردن فحش‌های جدید** به لیست  
- 🎙️ **انتخاب حالت خروجی** (متن / ویس)  
- 🔘 **کیبورد شیشه‌ای (Inline Button)** برای تعامل آسان  
- 💾 **ذخیره‌سازی تنظیمات** در فایل `config.json`  
- 🔢 **پشتیبانی از اعداد فارسی و انگلیسی**  
- ⏰ **اجرای ۲۴/۷** (بدون نیاز به خوابیدن)  

---

## 📌 نحوه استفاده  

1. فقط **سودوها** (کاربران مجاز) می‌توانند از ربات استفاده کنند.  
2. برای اضافه شدن به لیست سودوها، به آیدی‌های زیر پیام دهید:  
   - 📩 **@llxisagi**  
   - 📩 **@tcsrqil**  
   - 📩 **@ronaakoa**  
3. دستور **`/panel`** را در چت با ربات وارد کنید تا پنل مدیریت باز شود.  
4. از طریق دکمه‌های شیشه‌ای، تنظیمات را اعمال کنید.  
5. برای فحاشی، روی پیام یک کاربر **ریپلای** بزنید و بنویسید:  {تارگت} رو بگا {عدد از ۱ تا ۱٠٠}
   **مثال:** `مسعود رو بگا 5`
